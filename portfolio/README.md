# Portfolio Website

Personal portfolio built with React, Vite, and Mantine. Designed to showcase projects, experience, and a public resume in a clean, fast UI.

## Key Features
- Multi-page navigation with React Router: Home, Work History, Showcase, Resume, and Contact.
- Embedded resume viewer with quick “Open in New Tab” access.
- Project showcase routes, including a focused to-do app showcase page.
- Contact page for inbound inquiries.
- Legal/support pages included (Terms, Privacy, Reset Password).
- Mantine-based UI with custom theme and shared layout components.

## Tech Stack
- React + Vite
- Mantine UI
- React Router
